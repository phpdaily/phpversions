phpversions
===========

An API that expose all PHP versions.

## Usage

### Development

Install dependencies:

```bash
$ composer install
```

Use Symfony CLI to run server:

```bash
$ symfony serve
```

## About

This project was created to play with PHP 8.0 and Composer 2.
